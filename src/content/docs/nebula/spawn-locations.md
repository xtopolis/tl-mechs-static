Chests always spawn in one of a fixed set of cluster spots per zone — two chests per spawn, always at the same paired locations. Stones follow the same cluster rules. Within an hour, a cluster **cannot repeat**, so once you've seen a spawn you can rule that spot out for the rest of the hour.

## Seal Sanctuary

:::caution[12-Minute Window]
Seal Sanctuary opens and closes on a **12-minute cycle**. If you miss the open window, the players already inside will get the chest. Plan your approach accordingly.
:::

Seal's layout is visually repetitive, so rather than named clusters, scan all possible spawn points and watch for the blue beam.
